## Değişken İsimlendirme

- Camel Case 
- Type prefix

```
st,tx : string
nm : number 
dt : date
bo : boolean
bn : binary,byte
ln : integer
db : double

```
Ayrıntılı tür kullanılmayacak , number double , int, float alanlara karşılık gelir.
