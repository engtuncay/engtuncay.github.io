


# Installations

➖ pnpm installation

```bash
npm install -g pnpm
```

➖ degit installation

```bash
pnpm add -g degit
```

➖ global path of npm and pnpm packages

```bash
npm config get prefix        # npm'in global konumunu gösterir
pnpm config get globalDir    # pnpm'in global konumunu gösterir
```
